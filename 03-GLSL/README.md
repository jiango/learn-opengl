# Notes

GLSL 几个关键字： in, out, uniform
out 输出到下一个流水线
in 从上一个流水线输入
uniform cpu和gpu之间传递数据

类型：
vecn, bvecn, ivecn, uvecn, dvecn
n 为 2,3,4



glVertexAttribPointer的最后参数，偏移量是字节偏移量，本例子中为0和3*sizeof(float)
